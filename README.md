## Python

**Python** is a high-level, general-purpose language known for its simple, readable syntax. It is one of the most beginner-friendly languages in the world and is widely used across many fields.

### What It's Used For
- Data science and machine learning (NumPy, Pandas, TensorFlow)
- Web development (Django, Flask)
- Automation and scripting
- Artificial intelligence and research

### Why Learn It
- Easy to read — the code looks almost like plain English
- Huge community with tons of free resources
- Used in top companies like Google, Netflix, and NASA

### How to Get Started
- Download Python at [python.org](https://python.org)
- Use VS Code with the Python extension
- Try: `print("Hello, World!")`

### Useful Links
- [Python Official Docs](https://docs.python.org)
- [W3Schools Python](https://www.w3schools.com/python)

## JavaScript

JavaScript is the language of the web. It runs inside every browser and is the reason websites can be interactive — buttons click, pages update, animations play.

### What It's Used For
- Frontend web development (React, Vue)
- Backend development (Node.js)
- Mobile apps (React Native)
- Browser extensions and games

### Why Learn It
- The only language that runs natively in the browser
- You can build both frontend and backend with one language
- Massive ecosystem of libraries and frameworks

### How to Get Started
- No installation needed — open your browser's DevTools (F12) and start typing
- Or install Node.js to run JavaScript on your computer
- Try: `console.log("Hello, World!")`

### Useful Links
- [MDN JavaScript Guide](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
- [JavaScript.info](https://javascript.info)

## Java

Java is a robust, object-oriented language that has been a cornerstone of enterprise software and Android development for over 25 years.

### What It's Used For
- Android app development
- Large-scale enterprise applications
- Backend web services
- Desktop software

### Why Learn It
- Very structured — teaches good programming habits
- Still widely used in banks, governments, and large companies
- Strong job market, especially in enterprise and Android

### How to Get Started
- Download the Java Development Kit (JDK)
- Install IntelliJ IDEA (recommended IDE)
- Write your first program: `System.out.println("Hello, World!");`

### Useful Links
- [Oracle Java Tutorials](https://docs.oracle.com/javase/tutorial/)
- [W3Schools Java](https://www.w3schools.com/java)

## C++

C++ is a powerful, high-performance language that gives programmers low-level control over memory and hardware.

### What It's Used For
- Game development (Unreal Engine)
- Operating systems and system software
- Embedded systems (robotics, IoT devices)
- High-performance applications

### Why Learn It
- Teaches you how computers actually work at a low level
- Used in performance-critical software where speed matters
- Foundation for understanding other languages

### How to Get Started
- Install MinGW (Windows) or use g++ on Linux/Mac
- Use VS Code or Code::Blocks
- Write your first program: `cout << "Hello, World!";`

### Useful Links
- [cplusplus.com Tutorial](https://cplusplus.com/doc/tutorial/)
- [LearnCpp.com](https://www.learncpp.com)

## Rust

Rust is a modern systems programming language focused on safety and performance.

### What It's Used For
- Systems programming (operating systems, drivers)
- Web servers and command-line tools
- Game engines
- Blockchain and smart contracts

### Why Learn It
- Consistently voted the most loved language in Stack Overflow surveys
- Prevents entire categories of bugs before your code even runs
- Growing fast in systems programming and cloud infrastructure

### How to Get Started
- Install Rust via [rustup.rs](https://rustup.rs/)
- Use VS Code with the `rust-analyzer` extension
- Run `cargo new hello_world` to create your first project

### Useful Links
- [The Rust Book](https://doc.rust-lang.org/book/)
- [Rustlings exercises](https://github.com/rust-lang/rustlings)

## TypeScript

TypeScript is a superset of JavaScript that adds static typing.

### What It's Used For
- Large-scale frontend apps (Angular, React with TypeScript)
- Backend with Node.js
- Any project where JavaScript codebases become hard to maintain

### Why Learn It
- Catches errors before you run the code
- Makes code easier to understand and maintain in teams
- Used heavily in professional and enterprise projects

### How to Get Started
- Install Node.js, then run `npm install -g typescript`
- Create a file `hello.ts` and write: `console.log("Hello, World!")`
- Compile with `tsc hello.ts` and run with `node hello.js`

### Useful Links
- [TypeScript Official Docs](https://www.typescriptlang.org/docs/)
- [TypeScript for JavaScript Programmers](https://www.typescriptlang.org/docs/handbook/typescript-in-5-minutes.html)

## Go

Go (also called Golang) is a language built by Google in 2009. It is designed to be simple, fast, and efficient.

### What It's Used For
- Cloud and backend services (Docker and Kubernetes are written in Go)
- APIs and microservices
- Command-line tools
- Networking software

### Why Learn It
- Very fast compile times and execution speed
- Simple syntax — easy to pick up after Python or JavaScript
- Excellent for building scalable backend systems

### How to Get Started
- Download Go at [go.dev](https://go.dev)
- Use VS Code with the Go extension
- Write your first program: `fmt.Println("Hello, World!")`

### Useful Links
- [Go Tour (interactive)](https://go.dev/tour/)
- [Go by Example](https://gobyexample.com)
